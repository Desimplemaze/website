World's Tale

Imagine a place where students and teachers can collaborate or discuss history and their interest in history in a non-classroom environment. World's Tale is a site where students and teachers alike can learn about history in a new way other than reading from textbooks or video lectures. It is also a place where they can make up their own history timeline for the world and see what could have happened. They can make up their own World's Tale.


Technologies used: HTML/CSS, J2EE, Java, jQuery

Environment: Eclipse IDE for Java EE Developers

Animated Poseur/ Sprite Editor

The Animated Poseur is a simple tool for any users to create their own little characters (Sprites). The purpose of the Animated Poseur is to only create a simple sprite with the intention of using it in future projects.


Technologies used: Java, Swing Library

Environment: Eclipse IDE for Java Developers
